# Grant's Portfolio

Single-page portfolio website made for a class assignment.

## Features
- Mobile-first responsive design (2+ breakpoints)
- CSS Grid for projects section
- Flexbox for navbar and other layouts
- Responsive nav (stacks on mobile)
- Hover transitions on buttons and project cards
- Fade-in animation on sections
- Dark/light mode toggle (uses CSS variables)
- Accessible: one h1, visible focus styles, good contrast
